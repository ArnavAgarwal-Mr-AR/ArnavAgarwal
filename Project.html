  • Abdominal Trauma Detection

    Competition organised by RSNA USA.

    Related to Computer Vision Medicasl Research

    Interpreting CT scans for abdominal trauma, however, can be a complex and time-consuming task, especially when multiple injuries or areas of subtle active bleeding are present. This challenge seeks to harness the power of artificial intelligence and machine learning to assist medical professionals in rapidly and precisely detecting injuries and grading their severity. The development of advanced algorithms for this purpose has the potential to improve trauma care and patient outcomes worldwide.

  • Winning prediction in Korean Baseball League

    Data analytics in R

    Winning prediction calculation in games is the current real-world problem that we face. This project aims to make a model in R that predicts the winning probability of a Korean Baseball league based on the input data. There are already a few research papers available that use different machine learning approaches to solve the issue but we have tried to create a model, totally mathematical based, in R, platform-independent, sports-independent, and as handy as possible.

    GitHub

  • Aeronautical Networks

    Computer Networks using Python

    This project is made to solve a real-world problem of aerospace networking. We have to find the network packet route that a network packet will follow, to transfer data from network station to planes flying over the North-Atlantic Ocean. It is as interesting as it sounds. The dataset given to us was the current position of planes in the air. Our main goal is to find the route with the maximum data transmission rate and lowest end-to-end latency.
